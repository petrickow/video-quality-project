Exploring the examples using an IDE

This SDK includes project files for Eclipse and IntelliJ users in the root folder.

IntelliJ users

IntelliJ users can double-click examples.ipr to open the project. Please note: it may take a while for IntelliJ to index all the JavaScript files in the project.

Eclipse users

  1. Start Eclipse and choose File->Import...
  2. Choose Existing Projects into Workspace
  3. Click the Browse... button next to Select root directory and set it to the extracted archive root
  4. It should then pick up the examples project.
